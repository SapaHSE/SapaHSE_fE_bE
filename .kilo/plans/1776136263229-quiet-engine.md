# Plan: Fix Registration Issues

## Problem 1: Timeout Exception (CURRENT)
When clicking "Daftar Sekarang", there's a timeout error:
```
Unexpected error: TimeoutException after 0:00:30.000000: Future not completed
```

### Root Cause
The frontend is trying to call the backend at Railway (`https://sapahseon-production.up.railway.app/api`) but the request is timing out after 30 seconds.

This could be because:
1. Railway server is not responding
2. Network issue between Flutter app and Railway
3. The server is taking too long to process (maybe due to email sending)

### Solution Options

**Option 1: Increase timeout** (quick fix)
- Change timeout from 30s to 60s in `api_service.dart`

**Option 2: Skip email sending for testing** (recommended)
- Comment out the email sending code in AuthController temporarily
- This will help determine if the email sending is causing the timeout

**Option 3: Check Railway logs**
- Check Railway dashboard for server logs
- See if there's an error or server crash
