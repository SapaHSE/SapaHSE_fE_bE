# Plan: Refactor Profile Models and Fix Avatar Display

## Goal
Move duplicate models from `profile_service.dart` to the new `profile_model.dart`, remove unused fields, and add profile photo display.

## Steps

### 1. Update `profile_service.dart`
- **Remove** inline model classes (`ProfileData`, `UserLicense`, `UserCertification`, `UserMedical`) that are now in `profile_model.dart`
- **Add** import: `import '../models/profile_model.dart';`
- **Keep** the service methods (`getProfile`, `getLicenses`, etc.) and result wrapper classes

### 2. Update `profile_screen.dart`
- **Add** import: `import '../models/profile_model.dart';`
- **Remove** unused `_nik` field (redundant with `_employeeId`)
- **Keep** `_profilePhoto` (to be used)
- **Fix avatar display** - update CircleAvatar to show API profile photo as fallback:
  ```dart
  // Priority: local file > API photo > default icon
  backgroundImage: widget.avatarFile != null
      ? FileImage(widget.avatarFile!)
      : (_profilePhoto != null ? NetworkImage(_profilePhoto!) : null),
  ```

### 3. Add `http` package import for NetworkImage
- In `profile_screen.dart` add: `import 'package:http/http.dart' as http;` or use `dart:io` for cached network image
